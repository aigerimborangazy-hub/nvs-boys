# 🔥 Tinder Zodiac · NVS Boys Edition

Интерактивная веб-презентация в стиле Tinder для офисного праздника NVS Boys.

## 🚀 Деплой на Vercel (3 минуты)

### Шаг 1. Создай GitHub репозиторий
1. Зайди на [github.com/new](https://github.com/new)
2. Название: `nvs-boys` (или любое другое)
3. **Public** или **Private** — без разницы для Vercel
4. Нажми **Create repository**

### Шаг 2. Загрузи файлы
1. На странице репо нажми **"Add file"** → **"Upload files"**
2. Перетащи `index.html` (и `README.md` по желанию)
3. Внизу — **"Commit changes"**

### Шаг 3. Подключи Vercel
1. Открой [vercel.com/new](https://vercel.com/new)
2. **"Continue with GitHub"** → разреши доступ
3. Найди свой репо в списке → **"Import"**
4. Все настройки по умолчанию → **"Deploy"**
5. Через ~30 секунд получишь ссылку: `nvs-boys-xxx.vercel.app`

Готово. Любое изменение в репо автодеплоится.

## 📸 Загрузка фоток

Открой задеплоенную ссылку в Chrome/Safari → дойди до карточек парней без фото → жми **"TAP TO UPLOAD"** → выбирай файл. Фотки сохраняются в твоём браузере (localStorage), будут на месте при следующих открытиях с того же устройства.

**Важно:** localStorage привязан к домену. Загруженные фотки будут видны только в **том браузере и на том устройстве**, где ты их загрузила.

## 🎬 Структура

1. **Splash** — Tinder лого, START
2. **Watching** — карусель из 8 слайдов с фото/видео evidence
3. **Concept** — обращение к мальчикам
4. **Swipe** — 32 карточки парней с реальными свайпами
5. **Match** — анимация IT'S A MATCH с конфетти
6. **Pair Photo** — место под AI-фотку пары
7. **Final** — поздравление с Днём Защитника

## 🎮 Управление

- **Свайп влево/вправо** мышкой или пальцем
- **Кнопки ✕ / ★ / ♥** под карточкой
- **Стрелки клавиатуры** ← → для свайпа
- **Enter / Space** — продолжить на match/pair экранах
